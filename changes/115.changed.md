Class `EntityCollection` can now be accessed with `mammos_entity.EntityCollection`. Operations are moved into the `mammos_entity.operations` submodules and are no longer accessible from the `mammos_entity` namespace. (Internally, some private submodules were renamed: `_base` -> `_entity`, `_entities` -> `_factory`, and `_onto` -> `_ontology`. The code for `EntityCollection` has been moved to `mammos_entity._entity_collection`.)
