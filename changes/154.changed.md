The API for writing and reading files has been changed in a backward incompatible way. Writing is now done via methods `EntityCollection.to_csv`, `EntityCollection.to_hdf5` and `EntityCollection.to_yaml`. Reading with the functions `mammos_entity.from_csv`, `mammos_entity.from_hdf5`, and `mammos_entity.from_yaml`. The `io` submodule has been removed.
